## 2. 새로워진 주요 기능 ✨

### 📊 엑셀 형태의 빠르고 직관적인 재고 관리
- 기존의 복잡한 절차 없이, **엑셀처럼 입고(+) / 출고(-) 칸에 숫자만 입력**하고 탭(Tab) 키로 빠르게 이동할 수 있습니다.
- 상단 **[저장 및 동기화]** 버튼 한 번으로 모든 변경 내역이 일괄 저장됩니다.
- 재고 수량 변경 시 **'판매중', '단종', '일시 품절' 등의 판매 상태가 자동으로 변경**됩니다.

### 👥 강력해진 직원 권한 제어 (관리자 전용)
- 최고 관리자는 새로운 직원 계정을 무제한으로 생성할 수 있습니다.
- **재고 수정 권한 ON/OFF:** 특정 직원에게는 수량을 변경할 수 없도록 "조회 전용" 권한만 부여하여 휴먼 에러를 방지할 수 있습니다.
- 직원별 비밀번호 강제 초기화 및 퇴사자 계정 삭제가 가능합니다.

### 📅 한눈에 보는 기간별 통계
- 상단의 통계 버튼(주간, 월간, 연간, 지정 날짜)을 누르면, 기간 내 **누적 입/출고량**을 자동으로 계산하여 표 안에 나란히 보여줍니다.

### 📋 투명한 활동 로그 및 엑셀 다운로드
- [활동 로그] 메뉴에서, 언제/누가/어떤 품목을 얼마나 수정했는지 **100% 자동 기록**됩니다.
- 통계가 적용된 현재 리스트 상황 그대로 깔끔하게 포맷팅된 **[엑셀(XLSX) 다운로드]**를 지원합니다.

## 3. 대표적인 사용 방법 가이드

### [ 📝 입고 및 출고 일괄 처리 ]
1. 화면 검색창에서 제품 코드나 이름을 검색하시거나, 상단의 **카테고리 이동 버튼**을 클릭하여 해당 제품으로 이동합니다.
2. 제품 옆 **입고(+)** 칸이나 **출고(-)** 칸에 넣고 뺄 숫자를 입력합니다 (Ex. 입고 칸에 '10' 입력 시 기존 재고 + 10).
3. 필요한 경우 **판상 상태**를 변경하거나 **비고**란에 메모를 남깁니다.
4. 수정 사항 입력을 마쳤다면, 화면 상단의 파란색 **[입출고 저장 및 동기화]** 버튼을 반드시! 눌러 완료합니다.

### [ 👤 직원 계정 추가 및 권한 설정 방법 ]
1. 최고 관리자 계정(`admin`)으로 로그인합니다.
2. 대시보드 우측 상단 💜 보라색 **[직원 계정 관리]** 버튼을 클릭합니다.
3. 나타나는 창의 상단 박스에서 직위/이름/아이디/초기비밀번호를 입력하고 **[추가하기]**를 누릅니다.
   - 💡 꿀팁: 추가 시 하단에 있는 체크박스를 통해 *미리* 수정 권한을 부여할 수 있습니다.
4. **권한 회수 방법:** 하단의 등록된 직원 테이블 중앙 **[재고 수정 권한]** 열에서, 스위치가 **파란색이면 수정 가능**, 클릭해서 **회색으로 끄면 조회 전용**으로 즉시 변경됩니다.
