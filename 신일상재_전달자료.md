## 2. 주요 기능

### 📊 재고 현황 확인
- 전체 제품의 현재 재고 수량을 한눈에 확인할 수 있습니다.
- 제품별 입고/출고 내역이 자동으로 기록됩니다.

### 📥 입고 / 📤 출고 처리
- 제품을 선택하고 수량을 입력하여 입고 또는 출고 처리가 가능합니다.
- 처리 시 메모(사유)를 함께 기록할 수 있습니다.
- 모든 변경 내역은 자동으로 이력에 저장됩니다.

### 👥 직원 계정 관리 (관리자 전용)
- **직원관리** 버튼을 통해 직원 계정을 추가/삭제할 수 있습니다.
- 직원 계정은 관리자가 생성하며, 비밀번호도 관리자가 설정합니다.
- 직원은 입고/출고 처리만 가능하고, 계정 관리 권한은 없습니다.

### 📋 변경 이력 조회
- 누가, 언제, 어떤 제품을 몇 개 입고/출고했는지 전체 이력을 확인할 수 있습니다.

### 📁 엑셀 내보내기
- 현재 재고 현황을 엑셀(CSV) 파일로 다운로드할 수 있습니다.

## 3. 사용 방법

### 로그인
1. 위 **접속 URL**을 브라우저(크롬 권장)에서 열어주세요.
2. 아이디와 비밀번호를 입력하고 **로그인** 버튼을 누르세요.

### 입고/출고 처리
1. 대시보드에서 처리할 제품을 찾습니다.
2. **수량**을 입력합니다.
3. **입고** 또는 **출고** 버튼을 선택합니다.
4. 필요 시 **메모**를 작성합니다.
5. **저장** 버튼을 눌러 완료합니다.

### 직원 계정 추가 (관리자)
1. 상단의 **직원관리** 버튼을 클릭합니다.
2. **계정 추가** 버튼을 누릅니다.
3. 직원의 아이디와 비밀번호를 설정합니다.
4. **저장**을 눌러 완료합니다.
