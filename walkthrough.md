# 🚀 데일리하우징 네이버 블로그 자동화 구축 완료

대표님의 피드백을 모두 반영하여 **데일리하우징 네이버 블로그 자동화(반자동) 봇**의 뼈대 코드 작성을 완료했습니다! 🎉

## 🛠️ 적용된 기능 요약

1. **자동화 스케줄러 (월, 수, 금)**
   - 매주 월/수/금 오전 10시 정각에 서버가 깨어나 자동으로 블로그 글 작성 프로세스를 시작합니다.
2. **맞춤형 AI 프롬프트 적용 (ChatGPT-4o)**
   - 데일리하우징의 주력 상품(에디톤, 마루, 데코타일 등) 중 하나를 무작위로 선정해 집중적으로 다룹니다.
   - 대표님 요청대로 '셀프 인테리어'나 'DIY'는 배제하고, 무조건 **전문 시공의 퀄리티와 중요성**을 부각하도록 AI 명령어(Prompt)를 단단하게 고정했습니다.
3. **비공개(임시저장) 포스팅 로직**
   - 글이 곧바로 공개되지 않으며, 블로그 관리자의 **비공개(Secret)** 글로 우선 등록됩니다.
   - 대표님이 직접 네이버 블로그 앱을 열어 AI가 쓴 글을 쓱 훑어보시고, 현장 사진(Before/After) 2~3장만 추가하여 '공개 발행' 버튼만 누르면 완성됩니다!

## 🔑 다음 단계: API 키 발급 및 등록 가이드

이제 이 봇이 실제로 네이버와 통신하려면 대표님의 계정으로 발급받은 **열쇠(API Key)**들이 필요합니다. 당장 하실 필요는 없고, 여유로우실 때 다음 3가지를 준비해 주시면 제가 서버(Firebase)에 등록해 드리겠습니다.

> [!IMPORTANT]
> **준비해야 할 3가지 API 키**
> 1. **OPENAI_API_KEY**: ChatGPT 플랫폼에서 발급받은 API 비밀키
> 2. **NAVER_CLIENT_ID**: 네이버 개발자 센터(Open API)에서 앱 등록 후 발급받은 ID
> 3. **NAVER_CLIENT_SECRET**: 네이버 개발자 센터에서 발급받은 비밀번호

### 등록 방법 (터미널 명령어)
키가 준비되면 터미널(명령 프롬프트) 창을 열고 아래 명령어를 순서대로 입력하여 안전하게 저장하게 됩니다. (이 부분은 나중에 제가 도와드릴 수 있습니다.)

```bash
# 1. OpenAI 키 등록
firebase functions:secrets:set OPENAI_API_KEY

# 2. 네이버 클라이언트 ID 등록
firebase functions:secrets:set NAVER_CLIENT_ID

# 3. 네이버 클라이언트 시크릿 등록
firebase functions:secrets:set NAVER_CLIENT_SECRET
```

> [!TIP]
> 키를 모두 발급받으신 후에 저에게 **"블로그 자동화 API 키 다 발급받았어, 세팅해줘"** 라고 말씀해 주시면, 제가 안내에 따라 최종 세팅과 테스트 발행을 진행해 드리겠습니다!
